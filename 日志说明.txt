安装前提
node.js和git客户端
新装git需要配置邮箱和名称
git config --global user.name "Long H"
git config --global user.email "578440449@qq.com"
新机器需要配置ssh keys
检查用户目录下的.ssh目录下是否有id_rsa,id_rsa.pub,known_hosts三个文件,有则删除重新生成
ls -al ~/.ssh
ssh-keygen -t rsa -C "578440449@qq.com"(回车三连)
eval `ssh-agent -s`
ssh-add
clip < ~/.ssh/id_rsa.pub(这一段是复制ssh key，然后设置到你的github里)
测试是否成功
ssh -T git@github.com


(npm镜像在国外，安装可能会遇到网络连接超时问题
这个时候需要设置国内镜像npm config set registry https://registry.npm.taobao.org)
新机器需要利用npm安装hexo
npm install -g hexo
安装依赖包
npm install
创建一个空目录开始初始化
hexo init #构建hexo配置文件
新机器需要安装hexo-deployer-git
npm install hexo-deployer-git --save
常用命令
hexo g #完整命令为hexo generate,用于生成静态文件
hexo s #完整命令为hexo server,用于启动服务器，主要用来本地预览
hexo d #完整命令为hexo deploy,用于将本地文件发布到github上
hexo n #完整命令为hexo new,用于新建一篇文章
hexo clean 清理
hexo new "my new post" #发表一篇文章也可以手动创建md文档放到source\_posts目录下
最好下载一个markdownpad软件用来写文章


新增导航分类页面
hexo new page categories
